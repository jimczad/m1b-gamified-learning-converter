Monroe One BOCES Gamified Learning Tool Converter

Wayground, Gimkit, Nearpod, Microsoft Forms, and Edpuzzle are approved instructional tools within Monroe One BOCES that support interactive formative assessment and guided practice for students.

Educators may reuse their own instructional content from platforms such as Kahoot, Quizlet, and Blooket by converting it into an approved instructional tool using the workflow below.

Step 1: Get Your Questions Out of the Original Tool

Kahoot, Blooket, Gimkit

  • Open the quiz while logged in
  • Turn on Show Answers
  • Print → Save as PDF

Quizlet

  • If you own the set: Export → Copy text
  • If you do not own the set: Print → Save as PDF

Wayground (formerly Quizizz)

  • Open the assessment
  • Choose Worksheet view
  • Turn on Answer Key → Print or Download PDF

Step 2: Use the AI Converter

Upload your PDF or paste copied questions into the AI tool below. The AI will extract questions, answer choices, and correct answers into a neutral, transferable format.

👉 M1B Gamified Learning Tool Converter (opens in a new tab)

Once the AI finishes, copy the entire formatted output and return here.

Final Import Instructions

Gimkit

  • Create a new Kit
  • Select Import from Spreadsheet
  • Upload the exported CSV file

Wayground

  • Option A (Recommended): Use Wayground AI Import and paste the converted questions
  • Option B: Create a new Assessment and paste or import questions manually

Nearpod

  • Create a new lesson
  • Add a Time to Climb or quiz activity
  • Paste the formatted questions

Microsoft Forms

  • Create a new Quiz
  • Paste questions or use text import
  • Mark correct answers before releasing
