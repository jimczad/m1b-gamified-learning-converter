Step 1: Get Your Questions Out of the Original Tool

Most quiz platforms do not provide a clean export that includes questions and correct answers. The goal of this step is to make sure the text is visible and selectable so it can be accurately extracted by the AI tool.

General Rule (Works for Most Tools)

  • Open the quiz or assessment while logged in
  • Turn on Show Answers or the Answer Key
  • Select Print
  • When the browser’s print preview opens:
    • If the text is selectable, copy directly from the preview
    • You may also save as a PDF if the text is readable

Platform‑Specific Notes

Blooket

  • Do NOT save the PDF — image‑based export
  • Copy questions and answers directly from the browser’s print preview

Kahoot, Gimkit

  • Ensure Show Answers is enabled

Quizlet

  • Export → Copy Text if you own the set

Wayground

  • Worksheet view → Answer Key → Print
