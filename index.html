Educators may reuse their own instructional content from other platforms by extracting questions, then converting them into an approved tool using the workflow below.

Step 1: Get Your Questions Out of the Original Tool

Kahoot, Gimkit

  • Open the quiz while logged in
  • Turn on Show Answers
  • Print → Save as PDF

Blooket

  • Turn on Answer Key
  • Print the quiz
  • Copy directly from the browser’s print preview (do not save the PDF)

Quizlet

  • If you own the set: Export → Copy text
  • If not: Print → Save as PDF

Wayground (as a source)

  • Worksheet view → Answer Key → Print or Download

Step 2: Use the AI Converter

Upload a PDF or paste copied text into the AI tool below. It will extract questions, answer choices, and correct answers into a neutral format.
